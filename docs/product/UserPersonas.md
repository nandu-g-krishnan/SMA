# User Personas

## Quant Researcher
Designs features, tests hypotheses, and validates strategies.

## Discretionary Trader
Uses market intelligence, technical setups, and risk tools for swing/intraday decisions.

## Portfolio Manager
Monitors allocation, exposure, drawdowns, and sector concentration.

## Execution Operator
Supervises orders, broker connectivity, failures, and kill-switch events.

## Administrator
Controls users, permissions, secrets, environments, and audit reports.
