# Business Requirements

- Support Indian market trading across cash, futures, and options.
- Provide research, backtesting, walk-forward optimization, portfolio construction, and automated execution.
- Integrate Zerodha Kite Connect, SignalR/WebSockets, OpenAI, Python AI services, PostgreSQL, Redis, GitHub, and GitHub Actions.
- Support local native development and single-Ubuntu-VPS production for 1-3 users.
- Treat Docker and Docker Compose as optional convenience tooling, not mandatory runtime infrastructure.
- Preserve traceability from indicators, patterns, and strategies to source documents.
- Enforce risk controls before any order reaches the broker.
