# Domain Driven Design

## Market Data Bounded Context

**Aggregates:** Market Data Aggregate, Configuration, Run/Session.

**Entities:** Versioned records, calculations, status histories, and audit entries.

**Value Objects:** Identifiers, timestamps, money, quantity, confidence score, timeframe, regime, and source reference.

**Domain Events:** MarketDataCreated, MarketDataUpdated, MarketDataValidated, MarketDataFailed.

**Repositories:** Query by instrument, time range, version, owner, status, and source document trace.

**Services:** Validation, calculation, orchestration, permission checks, and monitoring publication.

**Specifications:** Completeness, freshness, risk eligibility, market-hours eligibility, and user authorization.
