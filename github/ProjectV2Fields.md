# GitHub Project V2 Fields

Status: LOCAL SOURCE OF TRUTH

GitHub Project V2 is a generated view of the local backlog. These fields must exist in the project before issue sync is considered complete.

| Field | Purpose |
| --- | --- |
| KnowledgeId | Traceability from issue to MasterKnowledgeBase |
| Knowledge Source | Filterable source class: TA_Workbook, Research_Paper, Kite_API, Market_Strategy, Architecture, Governance |
| Source Document | Original PDF, documentation, architecture, or governance source |
| Source Page | Source page or source section |
| Architecture Component | Owning architecture component |
| Initiative | Parent initiative |
| Epic | Parent epic |
| Feature | Parent feature |
| Story | Story identifier |
| Formula Verified | Yes/No validation for formulas and calculations |
| Backtest Verified | Yes/No validation for backtest completion |
| Paper Trade Verified | Yes/No validation for paper-trading completion |
| Live Verified | Yes/No validation for live validation |
| Compliance Status | Pass/Fail governance state |
| Priority | Delivery priority |
| Story Points | Relative effort |
| Status | Execution status |
| Model Version | AI/model governance identifier |
| Risk Review | Required review state for risk-sensitive stories |
